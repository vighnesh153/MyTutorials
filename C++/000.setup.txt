To add a cpp file, 
	Right click on 'Source Files' folder in the solution's explorer, 
	then add cpp file.

A project needs to have exactly 1 main function across all the cpp files.


To add a header file, right click on the Header files folder in the solution explorer 
and follow similar procedure.
